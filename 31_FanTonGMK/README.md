# FanTonGMK 方通公母口

NX C++ 方通公母口插件，用于改变两根方通交接方式：在公口方通端面生成凸舌，在交接方通上生成对应母口切除槽。

## 当前功能

- DLL 入口：`ufusr`
- 输出：`FanTonGMK.dll`
- UI：`FanTonGMK.dlx`
- 支持手动模式：选择公口面和母口面后生成一组公母口。
- 支持自动模式：选择一个基准端面后，扫描同平面方通并寻找交接方通批量处理。
- 支持两种轮廓：直角公母口、锥度公母口。
- 支持方通 R 自动识别，用于把公口高度和母口切深按外圆角修正。

## 使用建议

1. 选择“直角公母口”用于图中直上直下的交接。
2. 选择“锥度公母口”用于需要导入、防呆或更好装配余量的交接。
3. 公口宽度必须小于所选端面的有效宽度。
4. 母口间隙建议先用 `0.2 ~ 0.5` 试切，再按实际焊接/喷涂余量调整。

## 编译

手动编译：

```bat
build_manual\build.bat
```

Visual Studio 编译：

```text
FanTonGMK.vcxproj
```

默认 `UGII_BASE_DIR` 为 `D:\Program Files\Siemens\NX2412`，如果你的 NX 安装目录不同，请先在环境变量里设置 `UGII_BASE_DIR`。
